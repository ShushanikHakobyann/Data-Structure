# Flight Connection Optimizer

##  Description
This project is a flight network analyzer implemented in C++ using graph data structures.

Airports are represented as nodes and flights as directed weighted edges with:
- Cost
- Duration

The system simulates real-world airline route optimization using graph algorithms.

---

##  Features
- Cheapest route (Dijkstra’s Algorithm)
- Fastest route (Dijkstra’s Algorithm)
- Reachable airports within K connections (BFS)
- Critical airports detection (Articulation Points)
- CSV-based dataset loading
- Interactive CLI menu

---

##  Algorithms Used
- Dijkstra’s Shortest Path Algorithm
- Breadth-First Search (BFS)
- Depth-First Search (DFS)
- Graph Articulation Point Detection
